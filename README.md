Este é um projeto web interativo dedicado a **Ryomen Sukuna**, o principal antagonista do anime e mangá *Jujutsu Kaisen*. O site combina um design sombrio e imersivo com elementos interativos para explorar a história, habilidades e o impacto do Rei das Maldições.

---

## 🎯 Objetivo do Projeto
O principal objetivo deste projeto é criar uma landing page temática e responsiva que sirva como uma enciclopédia visual e interativa para fãs da obra. Além disso, o projeto foi desenvolvido como um desafio prático para consolidar conhecimentos em desenvolvimento web front-end (HTML5, CSS3 estruturado e manipulação de DOM com JavaScript).

## 🚀 Tecnologias Utilizadas
O projeto foi construído puramente com tecnologias web nativas (Vanilla Web), sem o uso de frameworks externos:
- **HTML5:** Estruturação semântica da página, organização dos blocos de conteúdo e incorporação de mídias (vídeos do YouTube).
- **CSS3:** Estilização avançada com temática *dark*, paleta de cores personalizada (tons de preto, vermelho amaldiçoado e dourado), design responsivo (Mobile First) e efeitos de transição/sombra.
- **JavaScript (ES6):** Implementação de interatividade, controle de estado de elementos visuais e manipulação de eventos para enriquecer a experiência do usuário.

## 🌟 Funcionalidades
- **Vídeo Incorporado Responsivo:** Player do YouTube perfeitamente integrado que se adapta a qualquer tamanho de tela (PC, Tablet ou Celular).
- **Design Temático Imersivo:** Estética visual inspirada no "Santuário Malévolo", utilizando sombras internas, bordas brilhantes e tipografia agressiva.
- **Seções Informativas:** Divisão clara entre a história da Era Heian, a explicação sobre os Dedos Amaldiçoados e suas técnicas principais (Clivagem e Desmantelar).
- **Interatividade Baseada em Eventos:** (Caso adicione scripts extras) Suporte para expansão de menus, revelação de curiosidades ou efeitos visuais ao passar o mouse.
